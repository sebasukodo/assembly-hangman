# Assembly Hangman

A small Hangman-style web game built with React and Vite.

The goal is to guess the hidden word before every programming language “dies”.  
With each wrong letter, one language gets eliminated — until only Assembly remains.

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Run Locally

```bash
npm install
npm run dev
````

## Build

```bash
npm run build
```
